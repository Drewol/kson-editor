# KSON Editor
Editor program for the Sound Voltex simulator format [KSON](https://gist.github.com/m4saka/a89594a17dc9422d75e01998bcfd2722)

## Done
* Chart loading
* Chart rendering
* Note & Laser placing
* Basic metadata editing
* Basic laser effects
