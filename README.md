# KSON Editor
Editor program for the Sound Voltex simulator format [KSON](https://github.com/m4saka/ksh2kson/blob/master/kson_format.md)

## Done
* Chart loading
* Chart rendering
* Note & Laser placing
* Basic metadata editing
* Basic laser effects
* Undo/Redo
* Curved lasers
* BPM editing
